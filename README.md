# Gemini Multi‑Gems Controller

여러 Gemini Gem을 한 화면에 나눠 띄워 **비교하고 전환하는 데 특화된 크롬 확장 프로그램**입니다.  
설치 후 바로 써볼 수 있게, 일반 사용자 기준으로만 정리했습니다.

---

## 이런 분께 딱 좋아요

- 같은 질문을 **여러 Gem에 동시에 던지고 비교**하고 싶은 분
- 역할이 다른 Gem을 **한 화면에 띄워 두고** 작업 흐름을 유지하고 싶은 분
- 번역/요약/코딩/아이디어 등 **병렬로 진행**하는 분

---

## 설치 방법 (처음 한 번만)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램 로드** 클릭
4. 이 프로젝트 폴더 `gemini-multi-gems` 선택
5. 목록에 추가되면 설치 완료

---

## 1분 사용법

1. 크롬에서 `gemini.google.com` 접속
2. 확장 프로그램 아이콘 클릭
3. **Gem URLs**에 Gem을 추가하거나, 아래 자동 가져오기 사용
4. **Layout Grid**에서 행/열 설정 (예: 2x2)
5. `Save & Reload` 클릭
6. 각 프레임 상단 드롭다운에서 Gem 선택

---

## Gem 자동 가져오기 (추천)

1. 첫 번째 프레임을 `https://gemini.google.com/gems/view`로 이동
2. 팝업에서 `Import from Gemini` 클릭
3. Gem 목록이 자동으로 추가됩니다

---

## 바로 써볼 만한 활용 예

- **리서치 + 요약**: 한 프레임에서 자료 수집, 다른 프레임에서 요약
- **코딩 + 리뷰**: 작성 Gem과 리뷰 Gem을 동시에 띄워 검토
- **번역 + 톤 조정**: 번역 Gem과 톤 수정 Gem을 병렬로 사용
- **아이디어 + 검증**: 아이디어 생성 Gem과 평가 Gem을 동시에 운영

---

## 알아두면 좋은 점

- 첫 번째 프레임은 기본적으로 `https://gemini.google.com/app`이 자동 로드됩니다.
- 레이아웃 변경(행/열)은 즉시 저장됩니다.
- Gem 목록은 드래그로 순서 변경이 가능합니다.

---

## 자주 묻는 질문

**Q. 모든 프레임이 동시에 로드되어 느립니다.**  
A. 사용하지 않는 프레임은 Gem을 선택하지 않으면 로드되지 않습니다.

**Q. Gem 목록이 안 보입니다.**  
A. `gemini.google.com/gems/view` 페이지가 첫 번째 프레임에 열려 있는지 확인하세요.

**Q. 레이아웃을 바꿨는데 적용이 안 된 것 같아요.**  
A. 팝업에서 행/열을 입력한 뒤 `Save & Reload`를 눌러주세요. 바로 적용됩니다.

**Q. Gem을 삭제했는데 다시 기본 Gem이 떠요.**  
A. 첫 번째 프레임은 기본적으로 `https://gemini.google.com/app`이 자동 로드됩니다.

**Q. 내 설정이 사라졌어요.**  
A. 설정은 브라우저 로컬에 저장됩니다. 브라우저 초기화나 다른 프로필을 사용하면 다시 설정해야 합니다.

**Q. 빈 화면만 보여요.**  
A. Gemini에 로그인되어 있는지 확인하고, 새로고침 후 다시 Gem을 선택해 보세요.
